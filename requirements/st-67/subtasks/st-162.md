# Configure Centralized Logging & Monitoring

Set up monitoring, metrics, and logging across all tenant environments, consolidated at a central account (per AWS Landing Zone model).

## Deliverables:

- CloudWatch or OpenSearch logging integration
- Centralized log aggregation account
- Metrics dashboard per tenant

## Acceptance Criteria:

- Deployment logs visible in central dashboard
- Alerts triggered on failed deployments or validation errors
- Monitoring verified for all 7 apps
