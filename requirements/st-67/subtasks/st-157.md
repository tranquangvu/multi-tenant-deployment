# Implement CI/CD Pipeline for Base Tenant Deployment

Set up the main deployment pipeline that builds, tests, and deploys code to the base tenant automatically.

## Deliverables:

- Bitbucket pipeline for base tenant deployment
- Deployment scripts and configuration templates
- Integration with Jira for build and deployment tracking

## Acceptance Criteria:

- Code changes auto-deploy to base tenant after merge
- Deployment status reflected in Jira ticket
- Base tenant environment verified to be functional
